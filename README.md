# IntelliJ IDEA Demo Project

这是一个用于演示IntelliJ IDEA功能的示例Java项目。

## 项目特点
- 使用Java 11
- Maven/Gradle构建工具
- JUnit单元测试
- 包含常用的Java编程示例
- 展示IntelliJ IDEA的高级功能

## 功能模块
1. 计算器功能 - 基本的数学运算
2. 字符串工具类 - 常用的字符串处理方法
3. 单元测试示例

## 如何运行
```bash
# 使用Maven
mvn clean compile
mvn test
mvn exec:java -Dexec.mainClass="com.example.demo.App"

# 使用Gradle
gradle build
gradle run
